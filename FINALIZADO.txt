🔧 COMO EXECUTAR

1. Instalar dependências:
   ```
   cd back/produtos && npm install
   cd ../catalogo-cestas && npm install
   cd ../pedidos && npm install
   ```

2. Executar cada servidor em um terminal diferente:
   ```
   Terminal 1: cd back/produtos && npm run dev
   Terminal 2: cd back/catalogo-cestas && npm run dev
   Terminal 3: cd back/pedidos && npm run dev
   ```

3. Abrir no navegador:
   └─ front/produtos.html
   └─ front/cestas.html
   └─ front/pedidos.html
